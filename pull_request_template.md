## 📝 작업 내용

- 작업하신 내용에 대해 설명해주세요. (Ex. 온보딩 페이지 퍼블리싱 진행 및 컴포넌트 구현)

## 📷 스크린샷

- 작업하신 부분에 대한 스크린샷 또는 동영상을 남겨주세요.
- 스크린샷 첨부 시 <img src="" width="20%" />처럼 width 옵션을 지정해주세요.

## 💬 리뷰 요구사항

- 리뷰어에게 전달하고 싶은 내용을 구체적으로 작성해주세요. (없다면 남기지 않아도 됩니다.)

## 📢 메모

- 진행한 작업과 관련하여 코드 리뷰 시 참고해야 하는 부분이 있다면 남겨주세요.

## ✅ PR Checklist

- PR이 다음 요구 사항을 충족하는지 확인하세요.

- [ ] 커밋 메시지 컨벤션에 맞게 작성했습니다.
- [ ] Reviewers 및 Assignees를 추가했습니다.
- [ ] 변경 사항에 대한 테스트를 했습니다. (버그 수정/기능에 대한 테스트)
